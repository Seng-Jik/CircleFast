# CircleFast v1.0.0 — 代码结构总览

## 项目信息
- 包名：com.vim.fasting
- 目标设备：OPPO Watch x3（ColorOS Watch，圆屏）
- APK 大小：Debug 682KB / Release 518KB
- 最低 SDK：API 26（Android 8.0）

## 文件清单（9个 Kotlin 文件）

```
com.vim.fasting/
├── MainActivity.kt              # 单入口 + 通知权限请求
├── data/
│   ├── FastingState.kt          # 状态模型
│   ├── FastingPreferences.kt    # 本地持久化
│   └── FastingTimer.kt          # AlarmManager 定时
├── notification/
│   └── NotificationHelper.kt    # 通知+振动
├── receiver/
│   ├── FastingAlarmReceiver.kt  # 到点回调
│   └── BootReceiver.kt          # 重启恢复
└── ui/
    ├── CircleFastScreen.kt      # 主界面
    └── CircularCountdown.kt     # Canvas 圆环
```

## 核心功能

| 功能 | 实现方式 |
|------|----------|
| 16h 倒计时 | Canvas 圆环进度条，每秒刷新 |
| 状态管理 | IDLE → FASTING → EATING 三态 |
| 通知提醒 | 16h 结束振动+通知，进食结束同样 |
| 省电 | 无前台Service，AlarmManager设完即退 |
| 重启保护 | BootReceiver自动恢复计时 |
| 圆屏适配 | Canvas手动绘制圆环+12个刻度 |
| 触摸 | 点击屏幕开始/停止断食 |

## 颜色方案（目前）
- 断食中：绿色 #4CAF50
- 进食中：橙色 #FF9800
- 空闲：灰色

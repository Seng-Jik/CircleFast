# CircleFast 使用说明书

> OPPO Watch x3 版 — 168 间歇性断食计时器

## 安装

### 前置条件
- OPPO Watch x3（ColorOS Watch）已开启开发者模式
- 手表已连接 Wi-Fi（与电脑同一网络）
- 电脑已安装 ADB

### 安装步骤

```bash
# 查看手表 IP（手表：设置 → 关于手表 → 状态信息）
adb connect 192.168.x.x:5555

# 确认连接成功
adb devices

# 安装 APK
adb install -r CircleFast-v1.0.0.apk
```

### 手表开启开发者模式

1. 手表：**设置 → 关于手表 → 连续点击「版本号」5次**

2. 手表：**设置 → 其他设置 → 开发者选项 → USB调试（打开）**

3. 如果要 Wi-Fi 调试：同一个位置开启「无线调试」

> **首次安装**：手表会弹窗确认是否允许安装，点「确定」。

## 使用说明

### 开始断食

在主界面点击屏幕中央 → 立即开始 16 小时断食计时。
- 同时会收到一条通知：「🔄 断食开始」
- 圆环开始以橙红色从底部顺时针转动
- 中央显示剩余时间
- 下方显示当前身体状态科普信息

### 断食期间看身体状态

每到一个时间节点，界面下方会更新科普内容：

| 时间 | 身体变化 |
|------|----------|
| 0h | 🔄 代谢切换 — 消耗糖原 |
| 1h | 📉 血糖平稳 — 饥饿感减弱 |
| 2h | 🧠 头脑清晰 — 酮体上升 |
| 4h | 🔥 脂肪燃烧 — 燃脂高峰 |
| 8h | 🛡️ 细胞自噬 — 修复受损细胞 |
| 12h | 💪 深度酮症 — 生长激素上升 |
| 16h | 🎉 断食完成 |

### 进食窗口

16 小时断食结束后：
- 自动切换为进食 8 小时倒计时
- 颜色变为绿色
- 收到通知：「🎉 断食完成」
- 进食期间也有科普信息（🍽️ 能量补充 / ⚡ 营养吸收 等）

### 手动停止断食

任何时候点击屏幕 → 结束当前断食。
- 会收到覆盖通知提醒
- 状态重置为待机

### 设备重启后

手表重启后会自动恢复计时。如果重启时断食已经结束，会自动进入进食窗口。

## 通知说明

通知仅在**三种情况**下发出一条：
1. **断食开始**（点击开始后）
2. **断食结束**（16 小时到点）
3. **进食结束**（8 小时到点）

- 每条新通知会覆盖上一条（最多只显示一条）
- 点击通知打开 App，不做其他操作
- **没有常驻通知 / 进度通知**，不影响手表续航

## 续航

CircleFast 对续航几乎无影响：
- 计时由系统 AlarmManager 处理（非 App 常驻后台）
- 每天耗电 < 1%（约等于收一条微信通知）
- App 纯黑背景，AMOLED 屏幕不发光部分不耗电

## 科普信息来源

当前科普内容基于间歇性断食常见的生理反应概括，非医学建议。如有疑问建议咨询医生。

## 常见问题

**Q：手环上找不到 App？**
A：在应用列表里找「CircleFast」或「断食计时」图标（纯黑背景 + 橙红弧 + 餐叉）。

**Q：通知不振动？**
A：检查通知权限：设置 → 应用管理 → CircleFast → 通知 → 允许通知和振动。

**Q：如何卸载？**
```bash
adb uninstall com.vim.fasting
```
